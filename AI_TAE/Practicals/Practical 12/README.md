# Practical 12: Study and Understanding of Adobe Firefly - AI-based Generative Tool

## Aim
To study and understand the working and applications of Adobe Firefly, an AI-based generative tool.

## Theory
Adobe Firefly is a generative AI tool developed by Adobe that allows users to create images, text effects, and designs using simple text prompts. It uses Artificial Intelligence to generate creative content such as:

- **Images**
- **Text styles**
- **Background edits**
- **Designs**

### Key Features
- **Text-to-Image Generation**: Converts text into images.
- **Text Effects**: Apply creative styles to text.
- **Generative Fill**: Add/remove objects in images.
- **Background Generation**: Automatically creates backgrounds.

### How It Works
Adobe Firefly uses:
- Machine Learning
- Neural Networks
- Large datasets

**Process**: User gives a prompt → AI processes → Output generated.

## Procedure / Steps
1. Open browser.
2. Go to Firefly website: [https://firefly.adobe.com](https://firefly.adobe.com).
3. Sign in using Adobe account.
4. Select feature (e.g., Text-to-Image).
5. Enter prompt (example: “A futuristic city at sunset”).
6. Click Generate.
7. Observe results.
8. Download output.

```
Text prompt: “A futuristic city at sunset”
```
